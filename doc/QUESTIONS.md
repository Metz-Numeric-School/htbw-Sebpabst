2. Comment faire pour qu'un nom de domaine pointe vers une adresse IP spécifique ?

- Je vais dans la gestion DNS chez OVH
- Je me connecte à mon espace client OVH, je vais dans "Domain Name", je sélectionne le domaine et je clique sur "Zone DNS"
- Je pointe vers le domaine principal, je choisi le type d'enregistrement et je défini mon adresse IP (192.168.23.141)
- Je me connecte à aaPanel avec les identifiants que l'on m'a généré, je vais dans "Website", dans "add site" et j'ajoute le nom de domaine.

4. Comment mettre en place un certificat SSL ?

Je me connecte à Aapanel, je vais dans "Website", sur mon site et dans les paramètres, Ensuite je vais dans l'onglet "SSL" et je fais "Apply"
